# Just a Paper ✍️

**究極にシンプルな執筆体験を、あなたのPCに**

画面に映るのは、たった一枚の白い紙だけ。余計なものは何もありません。

## このアプリについて

「Just a Paper」は、執筆することだけに集中したい人のためのテキストエディタです。

複雑な機能やボタンはありません。まるで本物の紙にペンで書くような、自然で心地よい執筆体験をお届けします。

## できること

**📝 お気に入りのフォントで書く**
- 明朝体：読書のような落ち着いた気分で
- ゴシック体：くっきりとした文字で集中力アップ  
- 等幅フォント：プログラマーやコード書きに最適

**📐 読みやすさを自分好みに**
- 行間の調整：狭め（0.8）から広め（2.0）まで、目に優しい間隔に
- 横幅の調整：50%から100%まで、画面サイズに合わせて最適化

**🎨 温かみのあるデザイン**
- PenCake風の優しい見た目で、長時間の執筆でも疲れにくい

## 使い始める方法

お使いのパソコンに Node.js（バージョン 20 以降）がインストールされている必要があります。

```bash
# 必要なファイルをダウンロード
npm install

# アプリを起動
npm start
```

## アプリを配布用にビルドする

```bash
# すべてのOS向けにビルド
npm run dist

# 特定のOS向けにビルド
npm run build -- --mac     # Mac用
npm run build -- --win     # Windows用  
npm run build -- --linux   # Linux用
```

## 開発者向け

コードの品質を保つためのツールも用意されています。

```bash
# コードのチェック
npm run lint

# コードの整形
npm run format
```

## 動作環境

- **Windows**: Windows 11 以降
- **Mac**: macOS Sonoma 以降
- **Linux**: Ubuntu 24.04 以降

## 性能について

- 起動は2秒以内のスピード感
- 10,000文字の長文でもサクサク動作（50FPS以上）
- インターネットがなくても使える

## 安全性について

このアプリは、あなたのプライバシーを最優先に設計されています。

- 書いた文章はすべてあなたのパソコンの中だけに保存
- インターネットに勝手に送信されることは一切ありません
- セキュリティ対策もしっかりと実装
